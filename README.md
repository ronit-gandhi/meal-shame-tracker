Overview

meal-shame-tracker is a Streamlit-based application for tracking daily food intake, calories, and user behavior, with a focus on accountability and simple analytics.

Objectives
	•	Track meals and calorie intake
	•	Provide daily summaries and trends
	•	Enable lightweight user interaction

Features
	•	Log meals with descriptions and calorie counts
	•	Daily summaries and totals
	•	Historical tracking and visualization
	•	Leaderboard-style comparison
	•	Optional commentary or feedback

Technology Stack
	•	Streamlit
	•	Supabase (database backend)

Repository Structure
	•	app.py: Main application
	•	database.py: Supabase integration
	•	utils.py: Helper functions

How to Run
	1.	Install dependencies
	2.	Configure Supabase credentials
	3.	Launch Streamlit app

Limitations
	•	Manual data entry
	•	Simplified calorie estimation
	•	Limited user authentication

Future Work
	•	Add barcode scanning or API-based food lookup
	•	Improve analytics and recommendations
	•	Add user authentication and profiles
	•	Mobile optimization

Purpose

This project demonstrates full-stack application development with a focus on usability, data storage, and simple analytics.
